# Project Context

This repository contains the solution for the FactorEnergia PHP Developer Technical Assessment (2026).

Time constraint: 2 hours.

The goal is not to build a runnable application but to demonstrate:

- Architecture thinking
- Clean code
- SQL reasoning
- API integration design
- Error handling and scalability awareness

Framework choice: Symfony.

Only code fragments and explanations are required.

## Tech stack

- PHP 8+
- Symfony conventions
- Doctrine-style entities
- SQL Server compatible SQL
- REST API integration
- Service-oriented architecture

## Deliverable format

Each section contains:

- Code fragments
- Short explanation of design decisions
